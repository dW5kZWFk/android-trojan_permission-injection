## English

This project was developed as part of my bachelor thesis (2022), which investigates the role of the Android permission system (Android API 23 to 32) in the functionality of malware.

A central aspect of the work is analyzing how trojans—introduced as malicious code into existing applications from the official app store—are able to obtain permissions. 
One method examined is the injection of permission requests directly into the original application, allowing the malicious components to inherit or request the required permissions.

This approach was analyzed using the open-source remote access tool AndroRAT as well as the Metasploit Android payload as practical examples and tested on applications from the google play store.

(The thesis was originally written in German and has not been translated. If you are interested in an English version of the full thesis, feel free to reach out.)

## German

Dieses Projekt entstand im Rahmen meiner Bachelorarbeit aus dem Jahr 2022. In dieser Arbeit wird untersucht, welche Rolle das Berechtigungssystem von Android (API 23 bis 32) für die Funktionsweise von Trojanern spielt.

Ein zentraler Aspekt ist die Analyse, wie Trojaner (die in bestehende Apps aus dem App Store injiziert werden) Berechtigungen erlangen können. Eine untersuchte Methode ist das Einfügen von Berechtigungsanfragen in die Original-App.

Dies wurde exemplarisch anhand der Open-Source-Remote-Access-Tools AndroRAT sowie des Metasploit-Android-Payloads untersucht.
